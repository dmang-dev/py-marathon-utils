# Bundled terminal fonts

Bitmap fonts used by `marathon_utils.terminals` to render Marathon terminal
screens. Generated from **Courier Prime** (SIL Open Font License 1.1) by
`scripts/generate_fonts.py`, matching the classic Courier12 metrics
(`MT 10 2 0 11 0`, `BB 10 12 0 -10 0`, 7px monospace advance).

- `Courier12.txt`, `CourierBold12.txt`, `CourierItalic12.txt`,
  `CourierBoldItalic12.txt` — generated bitmap fonts (this project, MIT)
- `OFL.txt` — license for the Courier Prime source font

Courier Prime: https://github.com/quoteunquoteapps/CourierPrime
